# CampusGo MCP Server 

## 版本信息
v0.1.0

## 产品描述

CampusGo MCP Server 是一个模型上下文协议(Model Context Protocol)服务器，为MCP客户端(如Claude Desktop、Cline、Cursor)提供与华为云服务CampusGo交互的能力。可以基于自然语言对CampusGo资源进行全链路管理。
